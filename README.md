# Interactive-story
(i will write in english because it is easier in context of coding and with my grammar level compared to my norwegian.)
This is my little interactive story i made based on inspo i found on pinterest (aesthetic wise) and other similar games ive played on itch.io. All the backrounds were found on pinterest or unsplash as normal photography and i edited the style and colour to match the 8 bit vibe i wanted. I found the svg to all the cards and boxes on figma for free, and i wrote all the story myself. I'm mostly satisfied enough with how it turned out, apart from the animations and transitions were i wish i couldve done more but it was more complicated than i initially thought and its not something you've taught us. 


Controls:
I decided to use 1,2 and 3 instead of arrow keys for simplicity due to there being 1 to 3 options available.
use R or ESC to go back to the start screen
Space does the same as 1, for a simpler way to go forwards when there is only one option. 
Backspace, as you might've guessed, goes back to the previous scene. 

Animations:
As i stated previously, animations were much more difficult than initially anticipated but i managed to include some very simple ones on both the start button and the different option buttons when you hover over them,  as i utilized "transition: all 0.3s ease;" which was reasonably easy to manage compared to keyframes and the other animations that required the use of both css and js. Despite this, i wish to learn it. I simply ran out of time.
